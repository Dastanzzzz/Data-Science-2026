## Deskripsi Repository

Repository ini dibuat sebagai portofolio pembelajaran mata kuliah Data Science dari Pertemuan 1 sampai Pertemuan 7. Repository ini berisi kumpulan notebook Jupyter Notebook (.ipynb) yang mendokumentasikan aktivitas praktikum, latihan, dan eksplorasi konsep dasar Data Science.

Tujuan dari repository ini adalah untuk menunjukkan proses belajar secara terstruktur, mulai dari pengenalan Python untuk Data Science, pengolahan data, eksplorasi data, visualisasi, hingga penggunaan library yang umum digunakan dalam analisis data. Setiap notebook berisi kode, output, dan kesimpulan singkat sebagai dokumentasi hasil pembelajaran.
